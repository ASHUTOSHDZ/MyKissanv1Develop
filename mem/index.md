# Project Memory

## Core
Mobile-first always. Most users are on phones — every screen must look great and work well at 360–414px before desktop.
Green & white palette. Primary green hsl(142 72% 32%), white background. No marigold/orange.
Bold sans-serif (Plus Jakarta Sans 800/900) for headings. No serifs, no italic-as-accent.
3D pressable UI: use .card-3d / .btn-3d / .press utilities for hover-lift and click-press feedback.
